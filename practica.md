# Ejercicios de Ecuaciones Cuadráticas

## 📌 Ejercicios

1. $$x^2 - 5x + 6 = 0$$

2. $$x^2 + 3x - 4 = 0$$

3. $$2x^2 - 7x + 3 = 0$$

4. $$x^2 - 9 = 0$$

5. $$3x^2 + 2x - 1 = 0$$

---

## ✅ Respuestas

1. $$x = 2 \quad,\quad x = 3$$

2. $$x = 1 \quad,\quad x = -4$$

3. $$x = 3 \quad,\quad x = \frac{1}{2}$$

4. $$x = 3 \quad,\quad x = -3$$

5. $$x = \frac{1}{3} \quad,\quad x = -1$$
